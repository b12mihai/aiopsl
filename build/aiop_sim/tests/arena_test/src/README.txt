===========================================
Introduction
===========================================
This file contains instructions for running the arena_test.
This test verifies several modules like: SLAB, DPNI, memory allocations and etc.
In order to run this test follow the instructions at "aiopsl/build/aiop_sim/apps/app_process_packet/src/README.txt".
Use aiopsl/misc/setup/arena_test_40.pcap instead of 
    aiopsl/misc/setup/eth_ipv4_udp.pcap in order to inject the packets.
